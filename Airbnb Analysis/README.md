## Airbnb Dataset Analysis

This project answers several business-related questions and analyzes the performance of Airbnb listings across major neighborhoods using a Power BI dashboard.


Data set- Airbnb Open Dataset available on [Kaggle](https://www.kaggle.com/datasets/arianazmoudeh/airbnbopendata)

Insights -
1.  What is the total number of Airbnb listings operating across different neighborhoods?

2. Total revenue generated from Airbnb listings.

3. Revenue contribution from room prices and service charges.

4. Number of Airbnb listings established over the years.

5. Average price trend across different neighborhoods.

6. Percentage of verified and unverified listings.

7. Availability of instant booking options.

8. Distribution of cancellation policies among listings.

9. Number of ratings received by different room types – hotel room, apartment, private room, and shared room.

10. Number of reviews received across different neighborhoods.
